About Giphy Search

Giphy Search is a lightweight, fast, and responsive web application that allows users to search and view trending and categorized GIFs powered by a Giphy API.

Giphy uses HTML, CSS, and JavaScript, so you can use:

  • A search bar with GIF results
  • A mobile-friendly design
  • A responsive contact form using Formspree to collect information

This project is ideal for learning how to work with APIs, and using JavaScript to practice making my webpages responsive.
